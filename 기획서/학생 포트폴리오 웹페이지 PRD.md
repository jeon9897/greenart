# 📘 학생 포트폴리오 웹페이지 PRD

## 1. 프로젝트 개요

학생들의 포트폴리오를 효과적으로 소개하기 위한 원페이지 웹사이트를 제작한다. 본 프로젝트는 안티그래피티(Antigraphy)를 활용하여 구현하며, 교육용 플랫폼에 적합한 신뢰감 있고 직관적인 UI/UX를 목표로 한다.

* **프로젝트 타입**: 원페이지 웹사이트 (One-page)
* **목적**: 학생 작품 전시 및 과정 홍보
* **타겟 사용자**: 예비 수강생, 기업 채용 담당자, 교육 관계자
* **디자인 컨셉**: 교육용 / 신뢰감 / 깔끔함 / 직관적

---

## 2. 디자인 가이드

### 🎨 컬러 시스템

* **Primary Color**: #2196F3 (메인 블루)
* **Secondary Color**: #E3F2FD (연한 블루)
* **Text Color**: #333333
* **Background Color**: #FFFFFF
* **Accent Color**: #FFC107 (포인트 강조)

### 🔤 타이포그래피

* **헤딩**: Bold / 24~48px
* **본문**: Regular / 14~18px
* **폰트 추천**: Noto Sans KR, Pretendard

### 🧩 UI 스타일

* 카드형 레이아웃
* 넉넉한 여백 (화이트 스페이스 강조)
* 부드러운 hover 애니메이션
* 라운드 처리 (border-radius: 8~16px)

---

## 3. 페이지 구조 (원페이지)

### 3.1 헤더 (Header)

* 로고 (좌측)
* 네비게이션 메뉴 (우측)

  * Home
  * 과정별 작품
  * About
  * Contact
* 스크롤 시 고정 (Sticky Header)

---

### 3.2 히어로 섹션 (Hero Section)

* 강렬한 메인 문구

  * 예: "학생들의 성장을 작품으로 확인하세요"
* 서브 문구
* CTA 버튼

  * "작품 보러가기"
* 배경 이미지 또는 간단한 애니메이션

---

### 3.3 과정별 작품 섹션

* 과정 카테고리 구분

  * 웹개발
  * 프론트엔드
  * 백엔드
  * 풀스택

#### 🔹 구성 방식

* 탭 또는 필터 기능
* 카드 UI로 작품 나열

#### 🔹 카드 구성 요소

* 썸네일 이미지
* 학생 이름
* 작품 제목
* 간단 설명
* 상세보기 버튼

#### 🔹 인터랙션

* hover 시 확대 효과
* 클릭 시 모달 또는 상세 페이지 이동

---

### 3.4 About 섹션

* 교육 과정 소개
* 강의 특징
* 취업 성과 간략 요약

---

### 3.5 Contact 섹션

* 문의 안내 문구
* 이메일 / 전화번호
* 간단한 문의 폼

---

### 3.6 푸터 (Footer)

* 학원 정보
* 저작권 표시
* SNS 링크

---

## 4. 기능 요구사항 (Functional Requirements)

### 필수 기능

* 원페이지 스크롤 네비게이션
* 과정별 작품 필터링 기능
* 반응형 웹 (모바일/태블릿/PC)
* 이미지 최적화
* 부드러운 스크롤 애니메이션

### 선택 기능 (추가 가능)

* 작품 좋아요 기능
* 검색 기능
* 관리자 업로드 기능 (CMS 연동)

---

## 5. 비기능 요구사항 (Non-functional Requirements)

* 페이지 로딩 속도 최적화
* SEO 기본 적용
* 접근성 고려 (a11y)
* 크로스 브라우징 지원

---

## 6. 기술 스택 제안

* **빌더**: 안티그래피티 (Antigraphy)
* **프론트엔드**: HTML, CSS, JavaScript
* **추가 가능**: React (확장 시)

---

## 7. 사용자 흐름 (User Flow)

1. 메인 진입
2. 히어로 섹션에서 관심 유도
3. 작품 리스트 탐색
4. 관심 작품 클릭
5. 상세 확인
6. 문의 또는 지원

---

## 8. 성공 지표 (KPI)

* 페이지 체류 시간
* 작품 클릭률
* 문의 전환율
* 재방문율

## 9. 향후 확장 방향

* 기업 연계 채용 시스템 구축

## 10. 참고 레퍼런스
